# homebrew-tether

Homebrew tap for [Tether](https://github.com/kmg/tether) — mobile tmux monitor with Claude Code activity detection.

## Install

```bash
brew tap kmg/tether
brew install tether
```

## Usage

```bash
tether start
```

Open the URL printed in your terminal.

## Background service

```bash
brew services start tether
```

## Updating

```bash
brew update
brew upgrade tether
```
